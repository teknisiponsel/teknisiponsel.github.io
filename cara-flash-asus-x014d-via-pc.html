3. Tolong bantuannya Bagaimana cara Install PC menggunakan Via Jaringan (LAN)?


Cara Sharing Internet Modem/Wifi Melalui Kabel Lan Artikel ini akan membahas Cara Sharing Internet Wifi/Modem Melalui Kabel Lan, maksudnya apa?, yaitu kita akan membahas tentang cara membagikan internet modem kita melalui kabel agar teman yang memerlukan juga dapat terkoneksi internet, ataupun ketika kita mempunyai koneksi wifi yang bisa internet, sedangkan teman kita tidak bisa, maka disini kita bisa membagikan koneksi wifi tersebut melalui kabel Lan.
Langkah2nya sangat mudah, pertama koneksi kan komputernya, lalu lakukan lah sharing, selesai deh :D
Tips Cara Mempercepat Koneksi Internet Gampang !
1. Koneksi kan komputer kamu ke internet melalui modem ataupun wifi, yang penting udah bisa internet, nama koneksi saya yang telah terkoneksi internet adalah Local Area Connection 2
2. Koneksi kan komputer kamu dengan komputer lain, jika belum bisa, saya sudah menyiapkan artikelnya, baca aja Cara Koneksikan Komputer ke Komputer Dengan Kabel UTP (Windows ke Windows). Saya telah menyambungkan komputer saya dengan komputer teman, nama koneksi nya adalah Local Area Connection 1.

3. Setelah itu kita akan melalukan sharing, klik Local Area Connection 2 (Koneksi yang terhubung ke internet), Lalu klik Properties, Pilih Tab Sharing yang diatas, lalu ceklis kedua kotak, dan pilih nama koneksi Local yang digunakan untuk menghubungkan kedua komputer, Klik OK OK trus.
4. Setelah selesai, seharusnya komputer satu lagi sudah bisa mengakses internet, jika tetap belum bisa maka coba matikan semua firewall.
Inti dari semuanya adalah, jangan salah memilih koneksinya, kamu harus tau koneksi mana yang bisa internet dan koneksi mana yang belum bisa internet.

7. bagai mana cara menscreen pada laptop asus ?


Menggunakan Tombol Print Screen

Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Selanjutnya, Gadgeter dapat menekan tombol prt sc sysrq yang berada di keyboard

Nantinya secara otomatis gambar screenshot akan dihasilkan

Kemudian, Gadgeter dapat membuka aplikasi Microsoft Paint atau Adobe Photoshop atau pengolah gambar lainnya

Setelah aplikasi terbuka maka Gadgeter tinggal melakukan paste dengan kombinasi Ctrl + V di keyboard

Nantinya gambar hasil screenshot akan muncul dan dapat langsung diedit atau disimpan dengan format jpg, png, bmp, dan masih banyak lagi di Harddisk.

Menggunakan Kombinasi Tombol Win + Print screen Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Kemudian, Gadgeter bisa menekan tombol berlogo Windows + prt sc sysrq secara bersamaan dalam waktu beberapa detik

Hasilnya secara otomatis akan disimpan dalam folder Screenshots yang berada di dalam folder Pictures yang bisa diakses dengan Windows Explorer

Menggunakan software Snipping Tool

Snipping Tool merupakan software atau tool bawaan yang sudah hadir dalam Windows. Dengan menggunakan tool ini, Gadgeter dapat melakukan berbagai macam model screenshot dibandingkan kedua cara di atas.

Pertama, Gadgeter dapat menentukan area yang akan di-screenshot pada layar laptop ASUS

Selanjutnya, Gadgeter dapat mencari software Snipping Tool di Start Menu

Setelah software ini dibuka, Gadgeter akan dihadapkan pada menu Snipping Tool

Untuk memulai screenshot, Gadgeter dapat memilih menu mode dan akan dihadapkan pada 4 mode sekaligus yang terdiri dari Free-form Snip, Rectangular Snip, Windows Snip, dan Full-screen Snip

Setelah memilih mode screenshot yang diinginkan maka Gadgeter dapat langsung melakukan screenshot

Hasilnya akan muncul di software Snipping Tool

Nantinya Gadgeter hanya tinggal memilih menu File > Save as dan hasil screenshot dapat langsung disimpan di folder yang telah ditentukan
