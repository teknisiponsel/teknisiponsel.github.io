Penjelasan dengan langkah-langkah:

Dalam menjawab soal tersebut, kita perlu mengingat beberapa aturan perhitungan pada bilangan kompleks. Berikut adalah langkah-langkah penyelesaian:

1. z1 - z2

z1 - z2 = (3 - 2i) - (5 + 4i)

= 3 - 2i - 5 - 4i

= -2 - 6i

Jadi, z1 - z2 = -2 - 6i.

2. z1 x z2

z1 x z2 = (3 - 2i) x (5 + 4i)

= 15 - 6i + 20i - 8

= 7 + 14i

Jadi, z1 x z2 = 7 + 14i.

3. z1/z2

z1/z2 = (3 - 2i) / (5 + 4i)

= [(3 - 2i) x (5 - 4i)] / [(5 + 4i) x (5 - 4i)]

= (15 - 2i - 12i - 10) / (25 + 16)

= (5 - 14i) / 41

Jadi, z1/z2 = (5 - 14i) / 41.

4. z2/z1

z2/z1 = (5 + 4i) / (3 - 2i)

= [(5 + 4i) x (3 + 2i)] / [(3 - 2i) x (3 + 2i)]

= (15 + 22i - 8) / (9 + 4)

= (7 + 22i) / 13

Jadi, z2/z1 = (7 + 22i) / 13.

5. z1^2

z1^2 = (3 - 2i)^2

= 9 - 6i - 6i + 4i^2

= 5 - 12i

Jadi, z1^2 = 5 - 12i.
