18. Gimana cara masuk ke akun lama woiiiiiii gua lupa kata sandi ​


Jawaban:

. Masukkan alamat email Anda pada laman login Gmail. Kemudian klik Next .

2. Saat diminta memasukkan kata sandi, klik Lupa kata sandi.

3. Gmail akan mengirimkan kode pengungkit ke akun email pemulihan Anda masukkan saat daftar gmail. Masukkan kode pengungkit tersebut lalu klik Next .

4. Anda akan diarahkan ke halaman pembuatan password Gmail baru. Setelah membuat password, klik Save Password.

Setelah Anda membuat kata sandi baru, Anda berarti telah berhasil login ke akun Gmail Anda.

Menggunakan Nomor Telepon Pemulihan

Semisal Anda tidak bisa mengakses akun alamat email pemulihan pada saat daftar Gmail, Anda masih bisa menggunakan telepon nomor yang pernah Anda daftarkan. Begini langkah-langkahnya:

1. Masukkan alamat Gmail Anda pada halaman login Gmail. Lalu klik Berikutnya .

2. Saat diminta memasukkan password, klik Forgot Password.

3. Pada tampilan untuk memasukkan kode pengungkit, klik Try another way .

4. Selanjutnya, Anda akan diminta untuk memilih penerimaan kode verifikasi melalui pesan atau telepon.

5. Cek ponsel Anda dan masukkan kode pengungkit yang dikirimkan pada kolom yang telah disediakan. Lalu klik Berikutnya .

6. Selanjutnya, Anda tinggal membuat password baru untuk login ke Gmail. Kemudian klik Simpan Kata Sandi . Setelah berhasil login, Anda akan melihat rekomendasi Security Checkup .

Pastikan rekomendasi keamanan tersebut telah Anda ikuti untuk menjaga keamanan akun Gmail Anda.

Penjelasan:

Jika Anda mengalami masalah di mana file yang ingin Anda upload ke Google Drive tidak dapat terbaca atau tidak terbuka dengan benar setelah diunggah, berikut beberapa langkah yang dapat Anda coba untuk memecahkan masalah tersebut:

1. **Periksa Format File**: Pastikan bahwa file yang ingin Anda unggah menggunakan format file yang kompatibel dengan Google Drive. Google Drive mendukung banyak format file, tetapi beberapa format mungkin tidak dapat dilihat langsung dalam tampilan pratinjau. Jika Anda mengunggah format file yang tidak didukung, pertimbangkan untuk mengonversinya ke format yang kompatibel.

2. **Unggah Ulang File**: Coba untuk mengunggah ulang file tersebut. Terkadang, kesalahan selama proses unggah dapat menyebabkan masalah. Pastikan bahwa proses unggah selesai tanpa gangguan.

3. **Baca Isi File**: Setelah mengunggah file, cobalah membukanya untuk memastikan bahwa kontennya masih utuh dan tidak rusak. Jika Anda menemukan masalah dengan isi file, perlu melakukan perbaikan pada file asli sebelum mengunggahnya kembali.

4. **Gunakan Aplikasi Google Drive**: Jika Anda mencoba membuka file yang rumit, seperti dokumen Office, spreadsheet, atau presentasi, cobalah membukanya menggunakan aplikasi Google Drive yang sesuai (Google Docs, Google Sheets, Google Slides). Google Drive sering kali lebih kompatibel dengan format filenya sendiri.

5. **Periksa Ukuran File**: Pastikan bahwa file yang Anda unggah tidak melebihi batasan ukuran yang ditetapkan oleh Google Drive. Jika file terlalu besar, Anda mungkin perlu membaginya menjadi beberapa bagian atau mengompresnya sebelum mengunggahnya.

6. **Periksa Koneksi Internet**: Pastikan bahwa koneksi internet Anda stabil selama proses unggah. Koneksi yang tidak stabil dapat menyebabkan gangguan saat mengunggah file.

7. **Periksa Izin Akses**: Pastikan Anda memiliki izin akses yang memadai untuk mengunggah file ke Google Drive. Jika file tersebut berasal dari sumber eksternal, seperti email atau penyimpanan cloud lainnya, Anda mungkin perlu memeriksa izin akses pada sumber aslinya.

8. **Gunakan Browser Lain**: Jika Anda mengunggah file melalui browser, coba gunakan browser yang berbeda. Terkadang, masalah dapat terkait dengan kompatibilitas browser.

9. **Perbarui Aplikasi atau Perangkat**: Jika Anda menggunakan aplikasi seluler untuk mengunggah file, pastikan bahwa Anda memiliki versi terbaru dari aplikasi tersebut atau perangkat lunak peramban yang Anda gunakan.

10. **Hubungi Dukungan Google Drive**: Jika Anda telah mencoba langkah-langkah di atas dan masih mengalami masalah, Anda mungkin perlu menghubungi dukungan Google Drive untuk mendapatkan bantuan lebih lanjut.

Dengan mengikuti langkah-langkah di atas, Anda dapat mencoba memecahkan masalah ketika file yang diunggah ke Google Drive tidak dapat terbaca atau terbuka dengan benar.

27. Bagaimana cara mengetahui password router netis jika lupa sandi, tanpa merestart router?


Cara mengetahui password router netis jika lupa sandi, tanpa merestart router adalah dengan membaca buku manual dari router tersebut. Selain itu, kita bisa mencari pada internet untuk pasword default dari router netis. Router bisa berupa perangkat jaringan komputer atau berupa sistem operasi seperti mikrotik.

Pembahasan

Router merupakan salah satu perangkat jaringan komputer. Router berfungsi untuk menghubungkan jaringan komputer yang satu dengan jaringan komputer lainnya. Router biasanya dilengkapi user dan password untuk mengamankan pengaturan dari router tersebut. Beberapa password default dari router netis adalah sebagai berikut

user admin password adminuser guest password guestuser netis password passworduser unknown password unknown

Dengan mengetahui user dan password default dari sebuah router kita bisa mengakses router dengan mudah. Oleh karena itu untuk seorang admin jaringan hendaknya merubah user dan password routernya sehingga tidak menggunakan password default lagi agar mudah diakses oleh orang lain.

Pelajari lebih lanjut

Demikian pembahasan mengenai cara untuk mengetahui password router netis, kita bisa membacanya pada buku manual atau mencarinya di internet. Untuk mempelajari materi lebih lanjut mengenai perangkat jaringan komputer lainnya dapat dibaca pada link berikut

Setelah itu,

1. Buka control panel.

3. Klik “Manage Another Account”.

4. Klik change the password (untuk mengganti password baru).

5. Masukkan password terbaru (cari password yang sulit dan mudah untuk diingat).
