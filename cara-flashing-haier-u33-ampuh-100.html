3. 3,6,9,12....U33 jawbanya


Barisan 3, 6, 9, 12


Maka diketahui

U1 = a = suku pertama = 3

U2 = suku ke-2 = 6


Tanya : U33 (suku ke-33)?


Jawab :

Pertama cari beda suku dari barisan tersebut,

b = U2 - U1

b = 6 - 3

b = 3


Lalu subsitusi nilai a dan b ke rumus U33

Un = a + (n - 1)b

U33 = 3 + (33 - 1)3

U33 = 3 + 29(3)

U33 = 3 + 87

U33 = 90


Jadi suku ke-33 adalah 90.

8. Jika U12 = 57 dan U5 = 22 Berapakah U33 = ?


Jawab:

162

Penjelasan dengan langkah-langkah:

Suku ke-n dapat dirumuskan :

Un = a + (n - 1) b

Diketahui :

U12 = a + (12 - 1) b57 = a + 11b

U5 = a + (5 - 1) b22 = a + 4b

Kita cari b nya dulu...

a + 11b = 57

a + 4b = 22

-------------------- ---

     7b = 35

       b = 35 / 7 = 5

Lalu cari a...

a + 4b = 22

a + 4(5) = 22

a + 20 = 22

a = 22 - 20

a = 2

Setelah itu baru kita cari suku ke-33

Un = a + (n - 1) b

U33 = 2 + (33 - 1) 5

U33 = 2 + (32) 5

U33 = 2 + 160

U33 = 162

13. Tentukan suku tengah dari informasi barisan aritmetika U1=-18 dan U33=10


Jawab:

Suku tengah dari barisan aritmatika tersebut adalah -4

Pembahasan

Rumus dari rumus tengah adalah

Ut = 1/2 (a+Un)

Maka, suku tengah barisan tersebut adalah

Ut = 1/2 (a+Un)

Ut = 1/2 (-18+10)

Ut = 1/2×(-8)

Ut = -4

Maka, nilai tengah barisan aritmatika tersebut adalah (-4)
