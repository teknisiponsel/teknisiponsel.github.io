11. bagaimana cara menginstal leptop


1. Nyalakan laptop, kemudian masuk ke menu bios dengan cara menekan tombol 'del' saat laptop mulai menyala.

2. Setelah masuk ke menu bios, pilihlah menu boot. Di sini kita harus memilih boot device dengan urutan pertama adalah CD/DVD ROM, dan save setting bios. Jadi pada saat laptop dinyalakan, otomatis akan membaca CD/DVD ROM yang sudah dimasukkan CD Windows, dan kemudian akan berlanjut cara instal laptop tahap selanjutnya ini.

3. Masukkan CD Windows ke dalam CD/DVD Rom, kemudian restart laptop.

4. Ketika laptop baru menyala, tunggu beberapa saat dan akan muncul tulisan "press any key bla bla bla..." tekan saja sembarang tombol di keyboard, (enter misalnya).

5. Proses loading... kemudian akan muncul tiga pilihan, enter saja untuk memilih To set up Windows XP now, press ENTER.

6. Kemudian akan muncul halaman license agreement, tekan saja tombol F8.

7. Kita asumsikan bahwa hardisk laptop sudah terpartisi biasanya dan seharusnya ada dua partisi, menjadi partisi C dan partisi D. Bila belum baca dulu: cara membuat partisi hardisk baru (cooming soon...).

8. Pilih pada partisi C dengan menggunakan tombol anak panah naik turun pada keywboard.

9. Selanjutnya akan muncul pilihan untuk melakukan format hardisk, pilih saja "format the partision using the NTFS file system, enter dan tunggu loading.

10. Sampai tahap cara instal laptop baru ini, kita tinggal menunggu saja. Akan ada proses format hardisk pada partisi C sampai proses selesai 100 persen, kemudian otomatis dilanjutkan copy file windows dari CD Windows, tunggu sampai proses copy selesai.

11. Saat proses copy file windows ke hardisk selesai, laptop akan otomatis restart, untuk mempercepat nya tekan saja tombol ENTER.

12. Setelah laptop restart, akan muncul kembali pilihan seperti tahap nomor 4, jangan tekan tombol apapun, biarkan saja dan otomatis proses instal laptop akan berjalan, dan kita hanya suruh menunggu sampai tahap selanjutnya.

13. Saat proses copy file Windows sudah selesai, tunggu sampai ada perintah "Regional and Language Option", pilih saja NEXT.

14. Selanjutnya ada pilihan untuk mengatur waktu (jam), pilih di menu dropdown nya : (GMT+7:00 Bangkok, Hanoi Jakarta) dan tekan NEXT.

15. Pada tahap cara instal laptop baru selanjutnya adalah, kita harus memasukkan Product Key, atau biasa kita sebut dengan serial number Windows, masukkan dan tekan NEXT.

16. Tahap selanjutnya adalah memasukkan nama kamu dan nama organisasi kamu, tulis saja terserah dan NEXT.

17. Tahap berikutnya mengisi computer name dan administrator password, abaikan saja atau isi tetapi harus diingat-ingat administrator password nya, NEXT.

18. Kemudian masuk ke menu Networking setting, biarkan saja seperti bawaan aslinya, tekan NEXT.

19. Selanjutnya muncul menu Workgroup or Computer Domain, biarkan saja seperti default Windows, tekan NEXT.

20. Selanjutnya kita tinggal menunggu saja, proses sedang berjalan.

21. Sampai muncul tulisan Display Setting "To improve the appereance... bla bla bla..." tekan saja tombol OK.

22. Muncul lagi tulisan Monitor Setting "Windows adjusted your screen... dst..." tekan tombol OK lagi.

23. Tunggu beberapa saat, dan tratatataa... Windows kita sudah siap sekarang dan proses cara instal laptop baru kita sudah selesai, dan kita bisa untuk memulai instal program lain yang diperlukan.

26. bang kalo custom rom tablet bisa ga sih, tapi gw liat di yt kok kebanyakan custom rom hp ya, bantu jawab ya ​


Jawaban:

Benar. Custom ROM adalah sistem operasi yang telah dimodifikasi oleh pengembang pihak ketiga dan dapat diinstal pada tablet untuk meningkatkan atau memodifikasi fitur dan performa tablet.

cara mengaktifkan custom rom

1. Persiapkan tablet dengan mengunduh ROM kustom yang diinginkan dan juga file Gapps (jika dibutuhkan) dari situs web yang tepercaya.

2. Masuk ke mode pemulihan dengan menekan dan menahan tombol volume bawah dan tombol power.

3. Arahkan ke mana file ROM yang telah diunduh dan tahan tombol volume atas untuk memilihnya.

4. Arahkan ke file Gapps jika perlu dan tahan tombol volume atas untuk memilihnya.

5. Tekan tombol daya untuk melanjutkan.

6. Pada tahap berikutnya, pilih opsi "Wipe Data/Factory Reset".

7. Setelah selesai, pilih opsi "Wipe Cache Partition".

8. Pilih opsi "Install" dan arahkan ke file ROM yang telah diunduh.

9. Tunggu sampai proses selesai.

10. Ulangi langkah 8 dan 9 untuk file GAPP jika dibutuhkan.

11. Setelah selesai, pilih Reboot System Now.

34. Cara menginstal debian


1.       Hidupkan komputer.
2.       Setelah itu masukkan DVD ke dalam DVDroom.
3.       Masuk ke bios dan ubah firstbootnya menjadi DVD.
4.       Lalu kita mulai menginstall debian berbasis text.
5.       Saat muncul tampilan Debian 6, kita pilih install untuk memulai instalasi.
6.       Pada pilihan language, kita pilih saja English.
7.       Lalu kita pilih area posisi kita, pilih saja Other.
8.       Saat muncul tulisan Continent or Region pilih saja  Asia.
9.       Kemudian kita pilih bangsa kita yaitu indonesai.
