12. Proses booting yang dilakukan jika komputer dalam keadaan hidup, yaitu dengan cara menekan tombol reset atau dengan merestart disebut dengan


Jawaban singkat:

warm boot

Penjelasan panjang:

Proses booting pada komputer ada 2 jenis, yaitu:

warm boot: disebut juga booting panas, dilakukan saat komputer dalam keadaan hidup dg cara menekan tombol reset.

cold boot: disebut juga booting dingin, dilakukan saat komputer dalam keadaan mati, caranya dg menekan tombol power.

Warm Boot

Booting itu ada 2 macam, warm boot sama cold boot, bedanya kalo warm boot itu booting yg dilakukan pas komputer nyala, sedangkan cold boot pas komputer mati.

23. Cara mereset komputer​


Jawaban:

Pertama, tekan tombol Start dan kemudian ketik “Remove“. Jika Anda menjalankan Windows 8.1, Anda akan melihat opsi untuk Remove everything and reinstall Windows pada pilihan pertama.

Selanjutnya, klik pada bagian Umum di bar vertikal di sisi kiri layar. Pilihan Remove everything and reinstall Windows akan muncul di sisi kanan layar. Klik Get Started untuk memulai proses.

Pengguna Windows 8.1 sekarang akan diminta untuk memasukkan disk instalasi Windows mereka sebelum melanjutkan. Namun, anda bisa mengabaikan langkah ini: hal pertama yang akan Anda lihat adalah peringatan bahwa jika melanjutkan akan menghapus tidak hanya semua aplikasim tapi semua yang telah diinstal pada laptop, termasuk semua file pribadi. Itu berarti semua dokumen, foto dan video, jadi pastikan Anda membackup data Anda sebelum Anda melangkah lebih jauh.

Jika Anda memiliki beberapa hard disk di komputer Anda, berikutnya anda akan ditanya apakah ingin menghapus semua drive atau hanya drive di mana Windows diinstal. Untuk mereset penuh, pilih opsi terakhir.

Selanjutnya, Anda akan ditanya apakah Hanya ingin menghapus file atau Bersihkan drive sepenuhnya. Jika Anda hanya mulai dari awal lagi, bisa memilih yangpertama. Jika anda ingin menyerahkan laptop lama Anda ke orang lain atau menjualnya, pilih opsi kedua. Meskipun ini membutuhkan waktu lebih lama, itu benar-benar akan menghapus data Anda, sehingga siapa pun yang berakhir dengan mesin tidak dapat mengambil file apapun dengan menggunakan perangkat lunak pemulihan.

Sekarang Anda akan melihat laman peringatan lain, memberikan Anda kesempatan untuk membatalkan atau melanjutkan. Ini adalah kesempatan terakhir Anda untuk mundur, jadi pastikan Anda telah diambil benar-benar mengambil semua yang Anda butuhkan dari hard disk mesin sebelum anda menekan tombol Reset.

29. Tuliskan alasan apa saja yang mengharuskan kita melakukan reset sebuah router


Jawaban:

Terlalu lama berjalan : Sama halnya seperti komputer, router anda kadangkala bisa macet karena ia terlalu “lelah” untuk berjalan, atau karena bug di firmware yang memakan terlalu banyak memori atau juga yang menyebabkan kepanikan kernel.Konflik alamat IP : Router anda mengelola alamat IP pribadi dan publik, dan terkadang juga bisa mengacaukannya. Jika dua perangkat di jaringan anda memiliki alamat IP yang sama, atau jika router Anda tidak memiliki alamat IP publik terbaru, koneksi mungkin akan rusak. Restart router akan me-reset tugas IP ini sehingga semuanya bisa bekerja kembali.Terlalu panas : Seperti komputer mana pun, router kita bisa menjadi terlalu pana, apalagi jika Anda menyimpannya di ruang tertutup untuk menyembunyikannya dari pandangan, hal tersebut bisa menyebabkannya macet.
