1.Cara menghidupkan komputer
 - Masukkan Steker yang menyambung ke CPU kedalam Stopkontak
 - Masukkan Steker yang menyambung ke Monitor kedalam Stopkontak
 - Tekan Power pada Monitor
 - Tekan Power pada CPU
 - Tunggu Loading Pembukaan selesai
 - Dan Komputer sudah dihidupkan

2.Cara membuka Folder di komputer, cara menyimpan file, cara mencari file di komputer
 - Klik Start (Dipojok kiri bawah), klik Folder yang mau dibuka, misalnya My Document atau My Computer atau My Video atau My Music. Jika ingin membuka Folder Biasa, tinggal klik 2x pada folder tersebut.
 - Klik File lalu Save, atau tekan tombol Ctrl + S, Tentukan lokasi file dan nama file, lalu klik tombol Save/
 - Klik Start (Dipojok kiri bawah) lalu klik Search, klik All files or Folder, isi Text Pertama dengan nama file lalu Text Kedua dengan format File, lalu klikText Ketiga lalu pilih lokasi yang ingin dicari, lalu klik Search

3.Tuliskan Cara membuka Word, Mengetik, dan Menyimpan File
 - Jika Wordpad, klik Start (Dipojok kiri Bawah) lalu klik All programs, lalu Klik Accessories, lalu klik Wordpad
 - Jika Open Office atau Microsoft Office, klik Start, lalu klik All Programs, lalu klik Open Office/Microsoft Office, lalu buka Open Office Word
 - klik pada Text lalu ketiklah sesuai dengan perintahmu di keyboard.
 - Klik File, lalu Save. Atau tekan tombol CTRL + S, lalu tentukan lokasi yang ingin disave, lalu ketik nama filenya, lalu klik Save.

13. Bagaimana cara membuka file fsb di android?


Jawaban:

Cara Membuka File FSB

Sebagian besar file FSB yang Anda temui dalam sebuah game mungkin dibuat dengan FMOD Designer. Anda dapat mengekstrak suara dalam file FSB menggunakan program seperti FSB Extractor atau Game Extractor.

Pengunduhan FSB Extractor sebagai file RAR. Anda akan membutuhkan program seperti PeaZip untuk membukanya. Kemudian, cukup pilih file FsbExtractor.exe untuk membuka alat.

Jika Anda lebih suka tidak mengekstrak data audio dari file FSB tetapi sebaliknya mendengarkan file secara langsung, Anda harus dapat melakukannya dengan menggunakan Music Player Ex. Anda mungkin perlu 7-Zip untuk membuka program ini karena setidaknya satu versi tersedia sebagai file 7Z.

Form-Z adalah program yang digunakan untuk membuka file FSB yang dikompilasi skrip. Ini paling mudah dilakukan dengan menyalin file FSB ke folder "skrip" folder instalasi program Form-Z. Plug-in harus siap digunakan setelah Anda me-restart program.

Jika Anda masih tidak dapat membuka file FSB Anda, Anda dapat memeriksa ekstensi file lagi untuk memastikan Anda tidak membingungkannya dengan file FXB, FS (Sumber F # Visual), atau SFB (Data Disk 3 PlayStation 3).

Jika Anda menemukan bahwa suatu aplikasi pada PC Anda memang mencoba untuk membuka file FSB tetapi itu adalah aplikasi yang salah atau jika Anda lebih suka program lain yang diinstal membuka file FSB, Anda dapat mengubah program default untuk ekstensi file tertentu.

15. Bagaimana cara mengembalikan file yang tidak bisa dibuka pada flashdisk?


Tahan tombol SHIFT di komputer/laptop dan masukkan flashdisk. Menekan dan menahan tombol SHIFT adalah untuk pengamanan saja, memastikan agar tidak ada program di flashdisk yang langsung dijalankan tanpa sepengetahuan pengguna (autorun.inf)Setelah itu hapus semua file dan folder yang merupakan shortcut (silahkan melihat kolom TYPE di windows Explorer). Pastikan yang dihapus adalah shortcut, dan jangan sampai salah melakukan klik dua kali (membuka shortcut) tersebut.Untuk mengembalikan semua data yang disembunyikan virus, bisa menggunakan program buatan saya Hidden File Tool atau langsung melalui Command Prompt. Silahkan memilih salah satu atau jika dengan Hidden File Tool ada yang tidak berhasil ditampilkan (misalnya nama file menggunakan huruf unicode: Arabic, China, Jepang dll) bisa ditambahkan menggunakan Command Prompt. Menggunakan Hidden File Tool, klik tombol Browse dan pilih drive flashdisk. Selanjutnya klik tombol -=SEARCH=-. Setelah hasilnya ditampilkan, pilih semua semuanya (klik file yang paling atas, tekan dan tahan SHIFT dan klik file yang terakhir). Selanjutnya klik tombol APPLY, sehingga semua file dan folder di flashdisk akan muncul semuaMenggunakan Command Prompt. Buka Command Prompt melalui Start Menu > Run dan tulis atau ketikkan:CMD lalu Enter. Atau melalui Start Menu > All Programs > Accessories > Command Prompt. Selanjutnya ketikkan huruf yang merupakan drive flashdisknya. Misalnya flashdisk ada di drive E, maka ketikkan E: (E diikuti tanda titik dua) dan tekan Enter. Setelah berhasil tampil E:\> di layar Command prompt, ketikkan perintah berikut: ATTRIB -S -H -R *.* /S /D kemudian tekan Enter.
