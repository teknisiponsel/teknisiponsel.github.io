1. Cara mengatasi bootloop


Light Bootloop Android
Permasalahan ini terjadi biasanya karena anda mengutak-atik sistem utama dan tidak menggunakan Recovery Mode, dan akibat dari light bootloop hanya bersifat sementara seperti blank screen.
Cara Mengatasi : Silahkan restart handphone anda terlebih dahulu, jika masih ada keanehan silahkan matikan dan cabut baterainya lalu pasang dan nyalakan lagi.

Medium Bootloop Android
Hal ini sering terjadi karena anda mencoba masuk ke system Recovery Mode dan menginstall script yang tidak jelas asal-usulnya.
Cara Mengatasi : Sebaiknya sebelum mengutak-atik apapun yang ada dalam Recovery Mode sebaiknya lakukan backup agar nanti bisa direstore jika terjadi masalah.

Hard Bootloop Android
Masalah ini adalah yang paling parah, dimana anda tidak akan bisa masuk kedalam menu handphone dan hanya stuck pada logo dan kemudian restart berulang kali.
Cara Mengatasi : Silahkan lakukan Factory Reset , jika anda ragu-ragu silahkan bawa ke service center.

Jika terjadi permasalahan pada smartphone android anda jangan terburu-buru bawa ke service, cari tau dulu masalahnya di google atau bertanya pada teman baru kemudian bertindak. Semoga cara memperbaiki hp android bootloop bermanfaat untuk yang membutuhkannya.

17. cara menginstal windows xp lengkap


Berikut ini merupakan tahapan dalam menginstal windows xp :

Pertama-tama, nyalakan komputer atau laptop kemudian tekan DEL pada komputer atau F2 pada laptop. Hal ini dilakukan untuk masuk ke setup menu serta memastikan "booting windows" dimulai dari CD.Masuk ke setup menu pengaturan "booting" dengan tampilan berbeda, tergantung pada merk masing-masing. Kemudian cari "boot" untuk "setting booting" pertama ke CD "installer windows XP". Arahkan CD ke posisi yang pertama, untuk memposisikan CD ke yang pertama biasanya terdapat keterangannya di samping kiri maupun bawahnya. Kemudian Exit dan Save, kemudian klik yes, maka komputer laptop akan segera booting. Namun perlu diingat, jangan sampai lupa memasukkan CD Windows XP ke komputer atau laptop sebelum restart lagi.Tekanlah tombol apasaja pada saat CMD muncul agar dapat masuk ke proses instalasi windows XP. Waktu yang diberikan untuk menekan tombol hanyalah 5 detik saja sehingga apabila tidak ditekan maka proses instalasi tidak akan terjadi.Masuk ke proses instalasi, pada saat ini tidak melakukan apapun dan kita hanya dapat mengamatinya saja.Tekan Enter untuk memulai proses instalasi.Tekan F8 untuk menyetujui proses instalasi.Kemudian masuk ke pengaturan partisi atau disebut juga pengaturan hardisk atau hard drive.Kemudian untuk meyakinkan penghapusan tekan L.Arahkan kursor ke partisi yang akan diinstal windows XP.Ketikan angka yang diinginkan atau tekan tombol C maupun tekan enter diperbolehkan jika ingin melewatinya.Tekan enter dan proses instalasi dimulai secara otomatis.Setelah melakukan langkah diatas, komputer atau laptop akan melakukan restart secara otomatis. Jangan menekan tombol apapun pada saat CMD muncul.Pada proses penyempurnaan instalasi windows XP, kita tidak perlu melakukan apapun.Setelah proses instalasi selesai, Ikuti langkah-langkah "Personalize name" sesuai data diri dan Setting windows XP sesuai dengan kode atau "product key" yang telah dipersiapkan. Setelah mengatur "Personalize name" dan kode, proses instalasi akan dilanjutkan.Kemudian kita dapat menunggu proses instalasi dan Windows XP telah selesai.Pembahasan

26. dimana letak perbedaan cara cold booting windows xp dengan windows 8 ?


Windows XP menggunakan metode Full Shutdown, artinya Komputer dengan OS Windows XP saat dinyalakan kembali membutuhkan waktu booting yang lama jika dibandingkan dengan Windows 8, ini dikarenakan saat komputer dimatikan tidak menyimpan driver dan kernel sesion dalam file hiberfil.sys

Windows 8 menggunakan metode Hybrid shutdown yang akan menyimpan driver dan kernel session di file yang bernama hiberfil.sys. Ketika kamu menyalakan PC lagi, maka Windows 8 menggunakan informasi tersebut untuk melanjutkan sistem yang sebelumnya telah berjalan, waktu booting pun lebih cepat daripada Windows XP

30. perbedaan windows 7 dan windows xp​


Jawaban:

Di bawah ini adalah 10 perbedaan utama antara Windows XP dan Windows 7.

1. Tidak Ada e-Mail Client

Outlook Express (OE) telah dihilangkan pada Windows Vista namun digantikan oleh Windows Mail. Anehnya, Windows tidak dilengkapi dengan e-mail client apapun saat pembelian. Pengguna harus membelinya lagi seperti Outlook atau menggunakan servis gratis seperti Windows Live Mail, atau download sebuah e-mail client yang open source seperti Thunderbird.

2. 32-bit dan 64-bit

Memang sebenarnya Windows XP juga memiliki versi 64-bit, namun tidak banyak orang yang mengetahuinya. Saat Anda melakukan upgrade dari Windows XP ke Windows 7, Anda harus memilih versi antara 32-bit atau 64-bit. Anda harus memilihnya berdasarkan hardware, driver, dan software yang Anda miliki.

3. Aero Desktop

Aero Desktop sebenarnya hanya mempercantik tampilan Windows 7 saja, Fitur seperti Aero Snap dapat memudahkan Anda untuk mengatur jendela-jendela yang terbuka dan tranparansi yang ada dapat memudahkan Anda untuk melihat di balik jendela-jendela yang sedang terbuka.

4. Documents and Settings

Pengertian dan Cara Mengatasi Bootloop di Android

Bambang Winarso - 12 January 2015

Bootloop adalah masalah pada piranti lunak Android yang ditunjukkan dengan tanda-tanda berupa kegagalan untuk masuk ke dalam sistem, ciri khasnya perangkat hanya menampilkan logo Android atau vendor secara berulang-ulang.

Jika perangkat Android mengalami bootloop, jangan panik apalagi menangis. Jangan buru-buru ke counter handphone dan membayar ratusan ribu, karena Anda pasti bisa kok mengatasi bootloop seorang diri. Bootloop sendiri ada dua macam yang dibedakan atas tingkat kerusakannya (berdasarkan pengalaman penulis, beberapa orang membedakannya menjadi tiga).

Pertamalight bootloop atau bootloop ringan. Biasanya bootloop ini hanya bersifat sementara, kadang-kadang saja kumatnya. Kedua, hard bootloop atau bootloop parah yang ditandai dengan kegagalan masuk ke sistem secara permanen walaupun sudah dicoba berulang kali.

Cara mengatasi bootloop Android

Setelah Anda ketahui apa itu bootloop dan jenisnya, sekarang kita akan coba mengetahui bagaimana cara mengatasi bootloop di Android.

Baik bootloop ringan ataupun berat sama-sama membutuhkan penanganan. Bootloop ringan walau Anda anggap tidak menggangu namun suatu saat akan menjadi parah yang tetap akan merepotkan. Jadi, ada baiknya untuk diperbaiki sebagaimana yang direkomendasikan.

Mengatasi bootloop ringan

Untuk memperbaiki perangkat yang mengalami bootloop ringan tidak diperlukan langkah-langkah teknis. Matikan perangkat Anda (jika hidup) kemudian cabut memori, kartu dan baterai. Kemudian diamkan selama 30 menit sampai 24 jam. Setelah itu pasang dan hidupkan kembali. Lalu amati, jika masalah bootloop masih terjadi maka lanjutkan ke solusi kedua.
