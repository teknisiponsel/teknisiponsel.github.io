Hai, sebelum kita mempelajari materi tentang “Operasi Hitung Bilangan Bulat”. Sebaiknya berdoa terlebih dahulu yaa, agar dimudahkan dalam memahami materinya. Sebelum lanjut ke pembahasan, apakah kalian tahu apa itu bilangan bulat?

» Bilangan Bulat adalah bilangan yang terdiri dari bilangan bulat positif (+) dan bilangan bulat negatif (-).

Contoh:

› Bilangan Bulat Positif = 1, 2, 3, 4, 5, ...

› Bilangan Bulat Negatif = -1, -2, -3, -4, -5, ...

[tex]\rightarrow[/tex] Memahami Rumus dalam Operasi Hitung Bilangan Bulat

Nah, untuk memahami rumus-rumusnya didalam operasi hitung bilangan bulat, kita pelajari terlebih dahulu operasi hitung penjumlahan dan pengurangan bilangan bulat.

Rumus:

[tex]\red{\boxed{(+) + (-) = (+) \: / \: (-)}}[/tex]

[tex]\red{\boxed{(+) + (+) = (+)}}[/tex]

[tex]\red{\boxed{(-) + (-) = (-)}}[/tex]

→ Intinya didalam operasi hitung penjumlahan dan pengurangan, kita melihat sebuah bilangan mana yang paling besar diantara bilangan bulat positif dan negatif. Jika ada yang Paling besar nilainya, maka itulah hasilnya.

Contoh:125 + (-50) = 75 (karena bilangan bulat positif lebih besar nilainya daripada bilangan bulat negatif)(-50) - 70 = -70  (karena bilangan bulat negatif lebih besar nilainya daripada bilangan bulat positif)

Nah, sudah tahu apa itu bilangan bulat begitupun dengan rumus-rumusnya? ayo kita lanjut ke pembahasan soalnya!

Pembahasan

Bilangan bulat adalah bilangan yang terdiri dari bilangan bulat positif (+) dan bilangan bulat negatif (-).

~Soal:

[tex]\tt \blue{125 + ( - 50) - 70}[/tex]

~Pembahasan Soal:

[tex]\tt \green{ = 125 + ( - 50) - 70}[/tex]

[tex]\tt \green{ = 75 - 70}[/tex]

[tex] \pink{\boxed{\bf = 5}}[/tex]

Kesimpulan:

Jadi, hasil dari 125 + (-50) - 70 adalah 5.
