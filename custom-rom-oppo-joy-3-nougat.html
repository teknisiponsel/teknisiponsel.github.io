18. Cara Mengupradge Android Jelly Bean Ke Nougat?


mungkin mengupgrade dari Android Jelly Bean ke Nougat tak semudah itu karena perlu waktu yang lama. jadi untuk mengupgradenya perlu:
1. koneksi internet stabil dan memadai
2. dukungan Android dari pembuat ponsel
3. dukungan upgrade lewat udara
4. WI-FI yang stabil koneksinya
5. baterai minimal 50% keatas
6. data wajib dibackup

jadi disini akan menjelaskan tentang upgrade melalui udara:
1. ke setelan => tekan pembaruan nirkabel => tekan check for updates
2. jika ada update terbaru, file akan terunduh. dan jika file sudah terunduh, kita akan diminta untuk menghidupkan ulang hp
3. ketika berada didalam update, tunggu hingga selesai dan jangan matikan ponsel. jika terjadi, ponsel akan mengalami kerusakan software (bootloop/brick)

kalau tidak bisa, dapat menggunakan file yang diunduh dari internet
1. jika updatenya pakai hp (langsung), pergi ke pilihan yang sama
2. kelocal updates lalu pilih file
3. jika ada peringatan seperti digambar kedua, pastikan bahwa file sudah benar dan lakukan update sekarang
4. tunggu hingga update selesai

28. bang kalo custom rom tablet bisa ga sih, tapi gw liat di yt kok kebanyakan custom rom hp ya, bantu jawab ya ​


Jawaban:

Benar. Custom ROM adalah sistem operasi yang telah dimodifikasi oleh pengembang pihak ketiga dan dapat diinstal pada tablet untuk meningkatkan atau memodifikasi fitur dan performa tablet.

cara mengaktifkan custom rom

1. Persiapkan tablet dengan mengunduh ROM kustom yang diinginkan dan juga file Gapps (jika dibutuhkan) dari situs web yang tepercaya.

2. Masuk ke mode pemulihan dengan menekan dan menahan tombol volume bawah dan tombol power.

3. Arahkan ke mana file ROM yang telah diunduh dan tahan tombol volume atas untuk memilihnya.

4. Arahkan ke file Gapps jika perlu dan tahan tombol volume atas untuk memilihnya.

5. Tekan tombol daya untuk melanjutkan.

6. Pada tahap berikutnya, pilih opsi "Wipe Data/Factory Reset".

7. Setelah selesai, pilih opsi "Wipe Cache Partition".

8. Pilih opsi "Install" dan arahkan ke file ROM yang telah diunduh.

9. Tunggu sampai proses selesai.

10. Ulangi langkah 8 dan 9 untuk file GAPP jika dibutuhkan.

11. Setelah selesai, pilih Reboot System Now.
