24. cara mematikan hp oppo A3s gimana ya?


cara mematikan hp oppo A3s yaitu :

Ada dua versi yaitu hard reset dan factory reset.

Hard reset :

1. tekan tombolo volume turun dan power secara bersamaan dalam beberapa detik

2. tekan tombol power dan pilih bahasa yang anda inginkan

3. tekan tombol power untuk menghapus data atau factory reset

4. tekan tombol power dan pilih yes

5. tekan tombol daya danpilih yes - reboot now

6. hp oppo A3s sudah diformat dan siap untuk digunakan


Factory reset :

1. aktifkan hp oppo A3s

2. arahkan ke menu settings

3. klik opsi factory reset

4. pilih reset device

5. pilih erase everything dan hp akan di-reboot

Pembahasan

Oppo A3s adalah varian handphone yang diproduksi Oppo dengan spesifikasi dan fitur yang sangat mengesankan. USP penuh,kamera ganda dan baterai besar merupakan keunggulan Oppo A3s. Kapasitas baterai 4.320 mAh dan RAM hingga 2 GB menambah kualitas Oppo A3s semakin bagus dan banyak diminati oleh pembeli. Sayangnya dalam hal reboot Oppo A3s agak susah dan harus melalui langkah yang cukup panjang.

Harga Oppo A3s relatif terjangkau yaitu 2,3 jutaan dan sudah tersedia di berbagai marketplace seperti Bukalapak,Lazada,Shoppe dan galeri handphone di sekitar anda.

33. jika kecepatan internet saya 4,5mbps dan total size file download saya 20gb, maka berapa lamakah waktu yang dibutuhkan untuk mendownload file tersebut?


Jawaban:

jika mengikuti logika maka jawabanya adalah 9,8763889 jam / 35555.5555556 detik

jika mengikuti soal maka jawabanyan adalah 4444.44444444 detik / 1,23455556 jam

Penjelasan:

mengikuti logika

4,5 mbps (megabit per second)

karna satuan dari provider itu Megabit maka kita harus rubah dahulu ke megabyte persecond

1 mbps (megabit persecond) = 0.125 megabyte (mb)

4,5 x 0.125 = 0.5625 megabyte per second

ukuran file 20gb

20gb = 20000mb

20000 : 0.5625 = 35555.5555556 detik

jika di rubah menjadi jam maka menjadi 9,8763889 jam

mengikuti soal

20gb = 20000mb

20000 : 4.5 = 4444.44444444 detik

atau 1,23455556 jam
