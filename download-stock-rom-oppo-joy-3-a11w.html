Kegunaan Custom ROM di Android. ... ROM atau Read Only Memory adalah jenis memori yang hanya bersifat “dibaca-aja” alias tidak bisa diinstalin ulang. Atau di-rewrite dengan file file baru. ROM sendiri berfungsi untuk menyimpan file-file atau data yang digunakan untuk menjalankan sebuah perangkat.

• What - Apa = Apa dibaca (Oppo).

• Why - Kenapa = Kenangapa.

• Who - Siapa = Sapa.

• When - Kapan = Kapan.

• Where - Dimana = Ing Ngendi.

• How - Bagaimana = Kepriye.
