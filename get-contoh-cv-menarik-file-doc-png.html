16. JPEG, TIF, PNG, GIF dll. Apakah ini tipe file​


Jawaban:

Ya

Penjelasan:

File JPG (Joint Photographic Experts Assemble) cocok digunakan untuk :

Gambar yang memiliki banyak warna, misalnya foto wajah dan pemandangan.Gambar yang memiliki gradien, misalnya perubahan warna yang perlahan-lahan dari merah ke biru.

File GIF GIF (Graphics Interchange Format) cocok digunakan untuk :

Gambar dengan jumlah warna sedikit (dibawah 256).Gambar yang memerlukan perbedaan warna yang tegas seperti logo tanpa gradien.Gambar animasi sederhana seperti banner-banner iklan, header, dan sebagainya.Print shoot (hasil dari print screen) dari program-program simple dengan jumlah warna sedikit.Gambar Grayscale memiliki palet 8-bit juga, jadi sah-sah saja/sebaiknya disimpan dalam format GIF.

File PNG (Portable Network Graphics) cocok digunakan untuk :

Gambar yang memiliki warna banyakGambar yang mau diedit ulang tanpa menurunkan kualitas.

28. Apa Perbedaan file jpg dan png​


Jawaban:

JPG menggunakan metode kompres lossy.

PNG menggunakan metode kompres lossless.

Penjelasan:

Jenis format atau ekstensi pada gambar

JPG / JPEG (Joint Photographic Expert Group)

GIF (Graphics Interchange Format)

PNG (Portable Network Graphics)

BMP(Bitmap)

TIFF(Tagged Image Format File)

Apa itu JPG / JPEG dan PNG ?

JPG  / JPEG atau Joint Photographic Expert Group merupakan skema kompresi file bitmap. Awalnya, file yang menyimpan hasil foto digital memiliki ukuran yang besar sehingga tidak praktis. Dengan format baru ini, hasil foto yang semula berukuran besar berhasil dikompresi sehingga ukurannya kecil.

by buku TI teknik komputer cyber indo

Jawaban :

file JPG menggunakan kompres lossy gunanya untuk mengoptimalisasi size foto dengan menghapus suatu bagian tertentu difoto tersebut.

sedangkan file PNG  menggunakan kompres losless yang berfungsi untuk foto dioptimalisasikan tanpa berkurangnya sama sekali kualitas.

Pembahasan

Word processor, atau dalam bahasa Indonesia disebut "pengolah kata", adalah jenis aplikasi atau program komputer yang menyediakan layanan pengolahan kata. Aplikasi jenis ini sangat penting digunakan dalam membuat beragam macam teks atau wacana, seperti dokumen, surat, catatan, undangan, formulir dan sebagainya.

Aplikasi pengolah kata umumnya menyediakan layanan membuat dokumen, memformat dokumen, menyunting/mengedit dokumen dan mencetak dokumen. Google Docs dan MS Word adalah dua jenis aplikasi pengolah kata yang saat ini paling populer digunakan. Google Docs sendiri adalah layanan pengolah kata daring (online) sehingga untuk mengaksesnya, kita membutuhkan koneksi internet. MS Word di sisi lain dapat tersedia secara luring (offline)

38. langkah langkah menyimpan file di aplikasi paint dengan format PNG!​


Jawaban:Cara pertama adalah menambah opsi simpan sebagai jpg ke toolbar akses cepat di bagian atas jendela Paint. Berikut adalah langkah yang perlu dilakukan:

1. Pilih File > Save as atau tekan tombol Ctrl + S, klik kanan pada format JPG, kemudian pilih Add to Quick Access Toolbar.

2. Anda akan mendapatkan ikon baru di ujung kiri atas layar dimana Anda dapat mengkliknya setiap saat untuk menyimpan gambar saat ini ke format jpg.

Cara kedua adalah memaksa Paint untuk menggunakan format gambar yang berbeda. Berikut caranya:

1. Buatlah gambar kosong dengan format jpg dan simpanlah ke sebuah lokasi di komputer Anda.

2. Klik kanan pada gambar tersebut, pilih Properties dan atur menjadi Read-only.

3. Tekan tombol Windows untuk membuka layar Start, dan ketik Paint untuk menampilkan hasil pencarian Microsoft Paint.

4. Pada bagian Target, gantilah nilainya menjadi berikut:

Pastikan Anda mengganti lokasi di atas dengan lokasi ke gambar kosong jpg yang telah Anda buat. Misalnya jika Anda menyimpan di folder Test, maka gantilah nama_user_anda menjadi Test.

Paint mulai sekarang akan selalu menampilkan gamabr tersebut ketika dimulai. Anda dapat menggunakan shortcut Ctrl + S untuk menyimpan file baru ke format jpg atau format lainnya. Namun perhatikan bahwa file yang dibuat tadi perlu diset sebagai read-only sehingga Anda tidak dapat menimpanya. Karena bersifat read only, menekan tombol Ctrl + S akan membuak dialog penyimpanan namun dengan tipe .jpg yang dipilih.
