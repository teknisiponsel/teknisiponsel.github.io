6. Diketahui IP Address dari sebuah komputer server adalah 192.168.10.1/29 Jumlah sisa IP yang bisa digunakan oleh komputer lain yang satu jaringan dengan komputer server adalah​


Diketahui IP Address dari sebuah komputer server adalah 192.168.10.1/29 Jumlah sisa IP yang bisa digunakan oleh komputer lain yang satu jaringan dengan komputer server adalah​  5 buah IP address.

Penjelasan dengan langkah-langkah

Diketahui :

IP Address dari sebuah komputer server adalah 192.168.10.1/29

Ditanya : Jumlah sisa IP yang bisa digunakan oleh komputer lain yang satu jaringan dengan komputer server

Jawab :

∑ porsi jaringan = 29

∑ porsi host = 32 - 29 = 3

Bit terbelakang = 1₁₀ = 00000001₂

Bit host = 001

Bit host(network) = 000 (ubah semua 1 jadi 0)

Bit terbelakang(network) = 00000000₂ = 0₁₀

Bit host(broadcast) = 111 (ubah semua 0 jadi 1)

Bit terbelakang(broadcast) = 00000111₂ = 7₁₀

Network address = 192.168.10.0

Broadcast address = 192.168.10.7

Banyak IP dalam server = 7 - 0 = 7

2 ip address dipakai untuk network address dan broadcast address, sedangkan sisa nya digunakan oleh komputer lain

Jadi IP yang tersisa yang bisa digunakan komputer lain yaitu sebanyak 5 buah IP address

20. Apakah pengertian dari database, server, proxy, ssh, dan ip :)


database, menurut saya adalah sekumpulan data yang telah tersimpan dan diatur, sehingga bisa dicari atau diambil debngan efisien ...
server, sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer ( kurang paham )
proxy, suatu mekanisme dan service serta jembatan anda ke facebook.com sekaligus memberikan konten yang sudah/bisa di cache/save di direktory proxy server untuk saat nanti jika anda mengakses ke facebook.com lagi tanpa harus mengambil langsung dari server facebook. 
ssh, (penjelasanya susah, tapi saya sering gunain ssh untuk gratisan)
ip, nama komputer anda dalam nilai angka di jaringan, komputer mengenali komputer lainnya dengan angka misal 192.168.1.100 bukannya nama dalam huruf seperti hostname atau domain seperti facebook.com. 

database = kumpulan informasi yang disimpan di dalam komputer secara sistematik untuk memperoleh informasi dari basis data tersebut.
server = sebuah sistem komputer yang menyediakan jenis layanan tertentu dalam sebuah jaringan komputer.
proxy = suatu server yang menyediakan layanan untuk meneruskan setiap permintaan kita kepada server lain di internet.
ssh = (Secure Shell) protokol jaringan yang memungkinkan pertukaran data melalui saluran aman antara dua perangkat jaringan.
ip = (internet protocol) suatu cara untuk mengetahui asal atau alamat suatu komputer berupa sistem penomoran masing-masing komputer yang bersifat unik atau tidak sama. Sistem penomoran itu sendiri terdiri dari empat bagian yang dipisahkan oleh titik contoh : 202.155.245.2

28. software yang diisntall pada workstation atau server yang berfungsi untuk menyaring IP mana saja yang bisa terkoneksi ke server adalah


Software yang kita install untuk menyaring IP aman saja yang bisa terhubung dengan server adalah Firewall. Firewall merupakan software yang memiliki fungsi untuk memilih IP mana saja yang bisa terhubung dengan Server. Dengan demikian, kita bisa menanggulangi adanya serangan ke server kita.

Pada sistem operasi windows, kita bisa mengaktifkan firewall melalui menu control panel dan memilih windows firewall. Selain menggunakan control panel, kita juga bisa mengakses windows firewall melalui menu services, kemudian cari servis dari windows firewall.

Pembahasan

Beberapa fungsi dari firewall serndiri adalah

Sebagai gerbang utama ketika server terhubung dengan clientSebagai pencatat semua log atau aktivitas yang terhubung dengan serverMelindungi data-data yang berada di server

Dengan demikian apabila komputer kita terhubung dengan komputer lain, bahkan komputer kita terhubung dengan internet, maka sudah semsetinya kita mengaktifkan firewall pada komputer kita agar komputer kita terlindungi.

Penjelasan:

Dynamic Host Configuration Protocol (DHCP) merupakan service yang memungkinkan perangkat dapat mendistribusikan/assign IP Address secara otomatis pada host dalam sebuah jaringan. Cara kerjanya, DHCP Server akan memberikan response terhadap request yang dikirimkan oleh DHCP Client.

Selain IP Address, DHCP juga mampu mendistribusikan informasi netmask, Default gateway, Konfigurasi DNS dan NTP Server serta masih banyak lagi custom option (tergantung apakah DHCP client bisa support).

Mikrotik dapat digunakan sebagai DHCP Server maupun DHCP Client atau keduanya secara bersamaan. Sebagai contoh, misalnya kita berlangganan internet dari ISP A. ISP A tidak memberikan informasi IP statik yang harus dipasang pada perangkat kita, melainkan akan memberikan IP secara otomatis melalui proses DHCP.
