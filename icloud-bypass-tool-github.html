Perbedaan tersebut dapat kita lihat melalui fungsinya.

1. Pick Tool
Pick Tool ini memiliki ikon seperti cursor mouse pada setting defaultnya. Berfungsi untuk menyeleksi atau memilih objek.

2. Shape Tool
Shape Tool ini berfungsi untuk mengedit garis atau objek dengan manipulasi titik.

Freehand Tool berfungsi untuk menggambar garis kurva atau garis lengkung dan juga garis lurus.

Zoom Tool ini berfungsi untuk mengubah tingkat pembesaran window document Corel yang sedang kita buka. Kita bisa mengedit objek yang terlalu besar atau objek yang terlalu kecil sehingga nantinya tammpilan tersebut akan sesuai di mata kita.

5. Text Tool
Text Tool ini berfungsi untuk menambahkan tulisan pada canvas Corel kita. Kita juga bisa mengatur jenis font, besar font, dan juga memperbesar atau memperkecil tulisan yang ada.

14. Untuk menyimpan dalam GitHub Anda harus a. menghubungkan dengan akun GitHub b. menggunakan akun lain C. menambahkan akun lagi d. menggunakan save as e. menyimpan kode​


Jawaban:

Untuk menyimpan dalam GitHub, kamu perlu melakukan langkah-langkah berikut:

a. Menghubungkan dengan akun GitHub: Pertama, kamu perlu membuat akun GitHub jika belum punya. Setelah itu, kamu harus menghubungkan akun GitHub tersebut dengan akun Git di komputermu. Hal ini bisa dilakukan dengan mengatur konfigurasi pada Git, seperti menambahkan alamat email dan nama pengguna GitHub ke dalam Git.

Setelah terhubung, kamu bisa membuat repositori baru di GitHub atau menggunakan repositori yang sudah ada.

b. Menambahkan akun lagi: Tidak perlu menambahkan akun lain untuk menyimpan di GitHub. Yang perlu kamu lakukan adalah menggunakan akun GitHub yang sudah terhubung dengan Git di komputermu.

c. Menggunakan save as: Save as tidak terkait dengan menyimpan kode ke GitHub. Save as digunakan untuk menyimpan file dengan nama baru atau di lokasi yang berbeda di komputermu, bukan untuk menyimpan ke repositori GitHub.

e. Menyimpan kode: Setelah terhubung dengan akun GitHub, kamu bisa menggunakan perintah-perintah Git seperti "git add", "git commit", dan "git push" untuk menyimpan kode ke repositori GitHub. Dengan melakukan commit dan push, perubahan kode yang kamu lakukan akan tersimpan di repositori GitHub.

Jadi, jawabannya adalah A. menghubungkan dengan akun GitHub.

17. bagaimana prinsip operasi bypass jantung


Kelas: XI
Mata Pelajaran: Biologi 
Materi: Sistem Peredaran Darah pada Manusia
Kata Kunci: Operasi Bypass Jantung


Jawaban pendek:

 

Prinsip operasi bypass jantung, adalah mengembalikan aliran darah normal ke arteri koroner yang terhambat, melalui pembuatan saluran di jantung yang baru menggunakan pembuluh sehat dari bagian tubuh lain.

 

Jawaban panjang:

 

Operasi bypass jantung, atau operasi bypass arteri koroner, digunakan untuk mengganti arteri yang rusak di otot jantung Anda. Seorang ahli bedah akan mengambil dan menggunakan pembuluh darah yang diambil dari daerah lain di tubuh untuk memperbaiki arteri yang rusak tersebut.

 

Operasi ini dilakukan saat arteri koroner tersumbat atau rusak. Arteri koroner ini adalah arteri atau saluran pembuluh darah yang memasok jantung dengan darah beroksigen. Jika arteri ini tersumbat, atau aliran darah di arteri ini dibatasi, jantung tidak dapat bekerja dengan baik. Hal ini bisa menyebabkan gagal jantung yang dapat berakibat kematian.


Dalam operasi, dokter akan membedah dan membuka dada, hingga jantung terlihat. Lalu saluran darah akan disambungkan ke mesin bypass kardiopulmoner. Mesin ini akan menyirkulasikan darah beroksigen melalui tubuh sementara dokter bedah melakukan operasi.

 

Dokter bedah juga akan menggunakan teknik pendinginan, untuk membawa suhu tubuh turun sampai sekitar 18 ° C. Teknik ini menunda proses tubuh dan membuat operasi jantung yang lama dapat dilakukan.

 

Dokter bedah kemudian akan mengeluarkan pembuluh darah yang sehat dari dalam dinding dada atau kaki. Pembuluh sehat ini diambil untuk mengganti arteri koroner yang tersumbat atau yang rusak.

 

Saat ahli bedah selesai, mesin bypass kardiopulmoner dilepaskan, dan fungsi bypass diperiksa. Setelah bypass bekerja, dada akan dijahit, dibalut, dan dibawa ke unit perawatan intensif (ICU) untuk pemantauan paska operasi.

22. Mengapa pada operasi bypass jantung dibuat saluran baru pada arteri?.


Jawaban:

Operasi bypass jantung adalah tindakan untuk mengatasi penyumbatan atau penyempitan pembuluh darah arteri koroner pada pasien penyakit jantung koroner. Prosedur ini dilakukan untuk mengalihkan fungsi arteri koroner yang rusak, menggunakan cangkokan pembuluh darah baru dari organ tubuh lain.

Jantung merupakan organ vital yang memompa darah ke seluruh tubuh, termasuk ke otot jantung itu sendiri. Pasokan darah ke otot jantung disuplai oleh pembuluh darah arteri koroner yang terbagi menjadi dua cabang utama, yaitu arteri koroner kanan dan kiri.

Arteri koroner ini dapat mengalami penyumbatan atau penyempitan akibat penumpukan plak pada dinding pembuluh darah (aterosklerosis). Arteri jantung yang mengalami penyumbatan dapat menimbulkan kerusakan otot jantung yang bisa berakhir dengan serangan jantung atau gagal jantung.

24. apa yang dimaksud bypass traffic​


Jawaban:

Ketika kita melakukan implementasi Simple Queue, dengan hanya berdasarkan target-address, maka Router hanya akan melihat dari mana traffic itu berasal. Sehingga kemanapun tujuan traffic nya (dst-address) tetap akan terkena limitasi. Tidak hanya ke arah internet, akan tetapi ke arah jaringan Lokal lain yang berbeda segment juga akan terkena limitasi.

Contoh :

IP LAN 1 : 192.168.10.0/24

IP LAN 2 : 192.168.11.0/24

Jika hanya dibuat Simple Queue dengan target-address : 192.168.10.0/24, traffic ke arah 192.168.11.0/24 juga akan terlimit. Agar traffic ke arah jaringan lokal lain tidak terlimit, kita bisa membuat Simple Queue baru dengan mengisikan dst-address serta tentukan Max-Limit sebesar maksimal jalur koneksi, misalnya 100Mbps. Kemudian letakkan rule tersebut pada urutan teratas no(0).

30. istilah pada github yang bisa digunakan untuk mengambil repostory dan remote adalah​


Jawaban:

JANGAN LUPA BINTANG 5 NYA YA KAK ^^

Penjelasan:

Istilah pada GitHub yang bisa digunakan untuk mengambil repository dan remote adalah:

1. Clone

Clone digunakan untuk mengambil (copy) repository Git menjadi local repository di komputer Anda. Caranya dengan mengetikkan perintah berikut:

`git clone url`

Dimana `url` adalah alamat URL lengkap repository Git yang ingin Anda clone.

Contoh:

`git clone https://github.com/namlnt/newrepo.git`

Git akan membuat folder `newrepo` lokal dan mengambil semua file dan versi yang ada di repository tersebut.

2. Fork

Fork digunakan untuk membuat salinan (copy) repository milik orang lain ke akun GitHub Anda. Caranya dengan menekan tombol "Fork" pada pojok kanan atas repository di GitHub.

3. Remote

Git menggunakan istilah remote untuk merujuk ke repository Git yang disinkronkan dengan local repository Anda. Remote bisa berupa repository GitHub atau server Git lainnya.

Remote bisa ditambahkan menggunakan perintah `git remote add` dan disinkronkan dengan perintah `git push` dan `git pull`.
