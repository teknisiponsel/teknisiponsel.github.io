4. Ada berapa cara membuka Software Programdi Windows 10?​


Jawaban:

Semua orang — termasuk Anda — mungkin sudah tahu bahwa semakin banyak program yang Anda install pada PC Windows 10 Anda, maka akan semakin lama pula waktu yang Anda butuhkan untuk masuk ke dalam Windows. Hal ini terjadi karena saat Anda menginstall banyak perangkat lunak, beberapa [program] di antaranya akan loading secara bersamaan dengan Windows, yang akhirnya dapat menunda proses booting dan membuat proses startup jadi relatif lemot.

Sama seperti pendahulunya, Windows 10 telah dilengkapi dengan fitur untuk menonaktifkan program startup atau aplikasi yang dimuat bersamaan dengan Windows. Cara untuk melakukannya adalah masuk ke dalam tab Task Manager –> Startup, kemudian Anda pun dapat melihat semua program yang dimuat secara bersamaan dengan Windows 10 lalu menghapus program agar tidak lagi dimuat secara bersamaan dengan Windows pada saat startup.

Selain menunjukkan program-program yang dijalankan pada saat startup, tab Startup pada Task Manager ini juga menunjukkan dampak program startup pada proses booting Windows. Sebaiknya, nonaktifkan program “berdampak tinggi” (high impact) untuk mempercepat waktu booting — kecuali jika program tersebut benar-benar diperlukan oleh sistem. Misalnya, jangan pernah sama sekali menghapus antivirus, firewall dan perangkat lunak keamanan lainnya dari daftar program-program startup.

 

Saat sedang menonaktifkan program-program startup, Anda bisa saja melihat program atau entry yang tidak pernah Anda dengar sebelumnya. Untuk memeriksa keaslian dari program yang tak dikenal tersebut, Anda bisa mengecek lokasi program-program tersebut pada PC Anda. Untungnya, menemukan lokasi dari program-program startup sangatlah mudah. Dengan menggunakan Task Manager saja, Anda bisa menemukan lokasi program startup tersebut. Berikut ini adalah cara-cara untuk melakukannya.

Menemukan lokasi program startup pada PC Windows 10

Untuk mengakses folder Startup, Anda bisa tinggal mengetik Shell: startup pada kotak Run command,

Langkah 1: Buka Task Manager. Ada banyak cara untuk membuka Task Manager di PC Windows 10, namun menggunakan shortcut Ctrl + Shift + Esc akan lebih mudah. Klik More details untuk melihat tampilan yang lebih lengkap.

Langkah 2: Sekarang, waktunya untuk pindah ke tab Startup

Langkah 3: Klik kanan pada program startup yang dimaksud, kemudian klik Open file location untuk membuka lokasi asli program pada PC Windows 10 Anda di File Explorer.

Selain metode di atas, Anda juga bisa mengkonfigurasi Task Manager agar selalu menampilkan path dari semua program startup. Untuk melakukannya, klik kanan pada kolom Name kemudian klik Command line untuk menambahkan kolom Command line di sisi paling kanan. Anda akan melihat bahwa kolom Command Line hanya menunjukkan lokasi (path) dari program. Dengan begini, Anda bisa langsung melihat lokasi dari program startup tersebut tanpa harus repot-repot membuka lokasi program itu.
