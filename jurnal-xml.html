12. Jelaskan Perbedaan format file PDF, HTML, XML dan XHTML


Jawaban:

PDF merupakan salah satu format dokumen digital yang banyak digunakan saat ini. Banyak orang lebih memilih menggunakan format ini karena lebih rapi dan tidak berubah saat dibuka menggunakan perangkat lainnya.

HTML atau Hypertext Markup Language adalah bahasa markup yang digunakan untuk membuat struktur halaman website agar dapat ditampilkan pada web browser.

XML atau Extensible Markup Language merupakan markup language yang diciptakan oleh World Wide Web Consortium. Bahasa ini digunakan untuk menyederhanakan pertukaran dan penyimpanan data.

XHTML adalah sebuah kombinasi dari HTML dan XML (Extensible Markup Language) dan merupakan bentuk sempurna (dalam arti, lebih ketat dan bersih dibandingkan HTML). Tujuan XHTML adalah menggantikan HTML.

14. .pdf, adalah .mobi, adalah ..epub, adalah .rtf,. adalah .doc, adalah .docx, adalah .html, adalah .xml adalah tolongin dongg ..


1. AZW– Amazon World. Sebuah format proprietary Amazon, yang menyerupai format MOBI kadang-kadang dengan dan kadang- kadang tanpa menyertakan Digital Rights Management (DRM). DRM pada format ini dikhususkan untuk Kindle Amazon.
2. EPUB– Electronic Publication. Format terbuka didefinisikan oleh Forum Open digital book dari International Digital Publishing Forum (idpf). EPUB mengacu kepada standar XHTML dan XML. Ini adalah standar yang sedang berkembang. baca juga cara membuat folder

Spesifikasi untuk EPUB dapat ditemukan di situs web IDPF, Adobe, Barnes & Noble, dan Apple, masing-masing memiliki DRM mereka sendiri. Format tersebut tidak kompatibel antara satu dengan yang lainnya. Saat ini sudah ada versi terbaru yaitu ePub 3, tetapi belum digunakan secara luas.

3. KF8- Format Kindle Fire dari Amazon. Hal ini pada dasarnya sama dengan prinsip ePub yang disusun dalam pembungkus Palm File Database (PDB) dengan Digital Right Management (DRM) milik Amazon.

4. MOBI– Format MobiPocket, ditampilkan menggunakan perangkat lunak membaca sendiri. MobiPocket tersedia pada hampir semua PDA dan Smartphone. Aplikasi Mobipocket pada PC Windows dapat mengkonversi Chm, doc, Html, OCF, Pdf, Rtf, dan Txt file ke format ini. Kindle menampilkan format mobipocket juga.

5. PDB- Palm File Database. Dapat menyertakan beberapa format buku digital yang berbeda, yang ditujukan untuk perangkat berbasiskan sistem operasi Palm. Pada umumnya digunakan untuk buku digital berformat PalmDOC (AportisDoc) dan format eReader juga.

6. PDF- Portable Document Format yang diciptakan oleh Adobe untuk produk Acrobat mereka. Format ini secara tidak langsung merupakan format yang digunakan untuk pertukaran dokumen. Dukungan perangkat lunak untuk format ini hampir mencakupi semua platform komputer dan perangkat genggam.

Beberapa perangkat memiliki masalah dengan PDF karena kebanyakan konten yang tersedia akan ditampilkan baik untuk format A4 atau surat, yang keduanya tidak mudah dibaca ketika diperkecil sesuai layar kecil. Beberapa aplikasi pembaca buku digital dapat menyusun ulang tampilan beberapa dokumen PDF, termasuk Sony PRS505, untuk mengakomodasi layar kecil.
baca juga cara memperkecil ukuran file PDF

7. PRC- Palm Resource File, Sering menyertakan alat baca Mobipocket tetapi kadang-kadang menyertakan eReader atau alat baca AportisDoc.

8. HTML- Hyper Text Markup Language adalah tulang punggung dari World Wide Web. Banyak teks yang didistribusikan dalam format ini. Selain itu, beberapa pembaca e-book mendukung Cascading Style Sheets (CSS) yang pada dasarnya gayautama panduan untuk halaman HTML.

9. CHM- Compressed HTML, sering digunakan untuk file bantuan Windows. Hal ini telah menjadi sangat populer untuk distribusi teks dan bahan pendukung lainnya melalui Web.

10. XHTML- versi khusus dari HTML dirancang agar sesuai dengan aturan konstruksi XML. Ini adalah format standar untuk data epub.

11. XML- tujuan umum markup language untuk pertukaran data. Dalam konteks digital book umumnya terbatas pada XHTML dan RSS feed meskipun beberapa format lain yang telah ditetapkan.

35. Apa saja kegunaan dari Microsoft Word dan apa keunggulan format file Microsoft Office berbasis XML?​


kegunaan dari Microsoft Word:

1. Menulis dan mengedit dokumen: Microsoft Word digunakan untuk membuat, mengedit, dan memformat dokumen teks seperti surat, laporan, esai, resume, dan banyak lagi.

2. Formatting dan tata letak: Word menyediakan fitur-formatting yang kuat untuk mengubah tampilan dan tata letak teks, seperti pemformatan huruf, paragraf, spasi, penomoran, pengaturan margin, dan lainnya.

3. Gambar dan grafik: Word memungkinkan pengguna untuk menyisipkan gambar, grafik, dan elemen visual lainnya ke dalam dokumen. Hal ini berguna untuk membuat dokumen yang lebih interaktif dan informatif.

4. Alat kolaborasi: Word menyediakan fitur kolaborasi yang memungkinkan beberapa pengguna untuk bekerja pada dokumen yang sama secara bersamaan. Hal ini memudahkan kolaborasi tim dan revisi dokumen secara efisien.

5. Pembuatan tabel: Word memiliki fitur untuk membuat tabel yang memungkinkan pengguna untuk mengatur dan menyusun data dalam format tabel.

6. Pemformatan halaman: Word memungkinkan pengguna untuk mengatur halaman dokumen, seperti header dan footer, penomoran halaman, pengaturan ukuran kertas, dan lainnya.

Keunggulan format file Microsoft Office berbasis XML, seperti .docx untuk Word, antara lain:

1. Kompatibilitas: Format XML Microsoft Office kompatibel dengan berbagai versi Microsoft Word, memungkinkan dokumen dapat dibuka dan diedit dengan lancar di berbagai platform dan versi perangkat lunak.

2. Ukuran file yang lebih kecil: Format XML memungkinkan kompresi data yang lebih baik, sehingga ukuran file dokumen lebih kecil dibandingkan dengan format file sebelumnya seperti .doc. Hal ini membantu menghemat ruang penyimpanan dan mempercepat waktu pengiriman dokumen.

3. Integrasi dengan aplikasi lain: Format XML Microsoft Office memudahkan integrasi dengan aplikasi lain dalam ekosistem Microsoft Office. Data dan konten dari dokumen dapat dengan mudah diekstrak dan digunakan dalam aplikasi seperti Excel dan PowerPoint.

4. Ketersediaan fitur canggih: Format XML memungkinkan penambahan fitur-fitur baru dan lebih canggih pada aplikasi Microsoft Office. Dengan format ini, pengguna dapat memanfaatkan fitur-fitur seperti pemodelan data, pemrograman makro, penyesuaian tata letak, dan banyak lagi.

5. Pemulihan dokumen yang lebih baik: Format XML mendukung pemulihan dokumen yang lebih baik dalam kasus terjadinya kegagalan atau kerusakan file. Format ini dirancang dengan metadata yang lebih lengkap, memungkinkan pemulihan yang lebih baik dari kerusakan atau kesalahan file.

Dalam keseluruhan, format file Microsoft Office berbasis XML memberikan keunggulan dalam kompatibilitas, ukuran file yang lebih kecil, integrasi dengan aplikasi lain, ketersediaan fitur canggih, dan pemulihan dokumen yang lebih baik.
