Pembahasan :

ROM atau Read Only Memory adalah sebuah bagian dari komputer yang menyimpan data yang bersifat permanen dan sekalipun mendadak mati komputernya maka file yang berada di ROM ini tidak akan hilang tetapi jenis file yang di simpan di sini hanya firmware.

ROM ini juga tidak memerlukan daya untuk menyimpan daya selain itu dibanding dengan RAM, ROM ini memiliki kecepatan yang lebih baik daripada semua media penyimpanan lain.

Sekarang kita bahas RAM , RAM atau Random Access Memory adalah sebuah  perangkat keras komputer yang berfungsi sebagai tempat penyimpanan data dan bersifat sementara Dan biasa juga dipakai sebagai pembaca data jadi semakin tinggi versi RAM yang dipakai maka semakin bagus kerja komputer anda
