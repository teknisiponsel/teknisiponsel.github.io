18. QUIZ MATEMATIKA Matematika Diskrit ^_^ ^_^ ^_^ Anda disuruh untuk membuat password suatu login pada sistem komputer yang panjangnya empat sampai sembilan karakter. Tiap karakter boleh berupa angka atau huruf (tidak dibedakan huruf besar dan huruf kecil). Buktikan bahwa banyak password yang dapat Anda buat untuk suatu login pada sistem komputer adalah 104.461.669.668.096 password!!!


Dengan huruf besar dan huruf kecil tidak dibedakan, maka dalam variasinya terdapat 26 bentuk alfabetik, serta tambahan berupa angka (dari 0 sampai 9) yang berjumlah 10 bentuk, sehingga dalam satu karakter memuat (26+10 = 36) kemungkinan karakter.

Untuk kasus 4 karakter, dengan tidak ada syarat khusus, banyaknya kombinasi:
n₄ = 36 x 36 x 36 x 36 = 36⁴

Hal tersebut serupa untuk 5 sampai 9 karakter, dengan:
nₓ = 36ˣ, untuk 5 ≤ x ≤ 9

Sehingga, jumlah kombinasi yang mungkin untuk sebuah password tersebut adalah:
n = n₄ + n₅ + n₆ + n₇ + n₈ + n₉
n = 36⁴ + 36⁵ + 36⁶ + 36⁷ + 36⁸ + 36⁹

Untuk mempermudah penghitungan, akan digunakan sifat deret geometri, dengan a = 36⁴, r = 36, dan n = 6, diperoleh:
n = S₆
n = 36⁴ (36⁶ - 1) / (36 - 1)
n = 1.679.616 x 2.176.782.335 / 35
n = 1.679.616 x 62.193.781
n = 104.461.669.668.096 kombinasi
