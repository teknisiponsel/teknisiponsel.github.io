4. Pada dasarnya, user harus mengunduh (download) dan menginstall paket Samba langsung dari ….  *Linux OSRepositoryHomepageSitusWindows OS​.


Jawaban:

Pada dasarnya, pengguna harus mengunduh (download) dan menginstal paket Samba langsung dari repositori Linux OS. Repositori Linux OS menyediakan berbagai paket perangkat lunak yang dapat diunduh dan diinstal melalui manajer paket yang disediakan oleh sistem operasi Linux. Samba adalah perangkat lunak sumber terbuka yang mendukung protokol jaringan Windows (SMB/CIFS) dan umumnya tersedia dalam repositori Linux OS.

Untuk mengunduh dan menginstal paket Samba, berikut adalah langkah-langkah umum yang dapat diikuti:

1. Buka terminal pada sistem operasi Linux Anda.

2. Perbarui daftar paket yang tersedia dengan perintah berikut (memerlukan akses root atau pengguna dengan hak akses administratif):

sudo apt update

atau

sudo yum update

3. Cari paket Samba dalam repositori dengan perintah berikut:

sudo apt search samba

atau

sudo yum search samba

4. Pilih paket yang ingin Anda instal, dan jalankan perintah instalasi. Misalnya, jika Anda menggunakan Ubuntu, Anda dapat menjalankan perintah berikut untuk menginstal paket Samba:

sudo apt install samba

atau jika Anda menggunakan Fedora, Anda dapat menjalankan perintah berikut:

sudo dnf install samba

Setelah proses instalasi selesai, Anda dapat mengonfigurasi dan menggunakan Samba untuk berbagi file dan printer antara sistem Linux dan Windows.

Penjelasan:

Penting untuk mencatat bahwa instruksi di atas hanya memberikan contoh umum dan dapat bervariasi tergantung pada distribusi Linux yang digunakan. Anda harus merujuk ke dokumentasi resmi distribusi Linux Anda atau mencari panduan instalasi yang relevan untuk mendapatkan instruksi yang tepat untuk sistem operasi Anda.

Pembahasan

Toolbox adalah kumpulan dari tool - tool atau perintah yang ada pada coreldraw. Pada setiap Toolbox terdapat tanda segiempat berlipat berwarna hitam yang menandakan ada tool lain yg tersembunyi. Tool yg tersembunyi ini dikenal sebagai “Flyout”.


Berikut ini beberapa flyout yang ada di coreldraw:

1. Pick Tool berfungsi untuk memilih objek dan mengatur objek.

2. Flyout Shape Tool berfungsi untuk memodifikasi objek yang telah dibuat, terdiri atas:

Shape : Mengubah bentuk garis, objek.Knife : Memotong objek.Eraser : Menghapus objek.Smudge Brush : Menghaluskan garis.Roughen Brush : Membuat garis kasar.Free Transform : Memutar objek dengan bebas.

3. Flyout Zoom berfungsi untuk mengadakan perubahan tampilan gambar dilayar, terdiri atas:

Zoom : Memperbesar/memperkecil tampilan.Hand : Menggerakkan objek dengan mendorong.

4. Flyout Freehand, terdiri atas:

Freehand : Menggambar garis bebas.Bezier : Membuat garis dengan titik-titik hubung.Artistic Media : Membuat garis artistik.Pen : Membuat garis lengkung, kurva dengan hubungan titik-titik.Polyline : Menggambar objek pola tertentu.3 Point Curve : Menggambar kurva.Interactive Connector : Menggambar garis hubung.Dimension : Membuat garis keterangan lengkap dengan keterangan panjang garis.

5. Flyout Rectangle digunakan untuk membuat gambar berupa kotak, terdiri atas:

Rectangle :Menggambar persegi panjang.3 Point Rectangle : Menggambar persegi panjang dengan 3 titik.

Pembahasan

Asumsi awal: Sistem operasi yang digunakan adalah Linux.

Pada sistem operasi Linux, untuk menambahkan user, terdapat 2 buah perintah yang dapat digunakan, yaitu [tex]\tt adduser[/tex] dan [tex]\tt useradd[/tex].
Pada beberapa distro Linux, [tex]\tt adduser[/tex] adalah symbolic link ke [tex]\tt useradd[/tex]. Namun, tidak demikian pada beberapa distro Linux lainnya.

adduser merupakan script Perl untuk menambahkan user secara “lebih aman”, karena tanpa parameter apapun selain username, [tex]\tt adduser[/tex] akan membuat home directory untuk user tersebut sesuai username, menyalin file-file default untuk user yang dispesifikasikan pada [tex]\tt /etc/skel[/tex], dan menampilkan wizard untuk melengkapi informasi user.useradd merupakan perintah low-level untuk menambahkan user. Tanpa parameter flag/opsi apapun, [tex]\tt useradd[/tex] hanya akan menambahkan user saja, tanpa melakukan hal lain yang dilakukan secara default oleh perintah [tex]\tt adduser[/tex].

Detail tentang sintaks dan cara penggunaan kedua perintah dapat dibaca pada help atau manualnya.

PENYELESAIAN

Yang akan kita lakukan adalah membuat user dengan rincian:

nama (username) = tooldirektori (home directory) = /home/toor123shell (login shell) = bash
Untuk shell, kita harus memberikan lokasi absolut [tex]\tt bash[/tex] yang terdapat sistem. Pada umumnya, file binary [tex]\tt bash[/tex] terdapat di /bin/bash, atau /sbin/bash. Tidak menutup kemungkinan terdapat di lokasi lain pada Linux yang sudah customized.
Lokasi absolut ini dapat ditentukan dengan perintah [tex]\texttt{which bash}[/tex].
Oleh karena itu, selain menggunakan lokasi absolut untuk login shell, agar tidak salah, kita dapat menggunakan [tex]\texttt{\$(which bash)}[/tex].UID (user ID) = 110GID (group ID) = 110komentar = "user ini baru lho"​

Cara 1: Dengan perintah “adduser”

Parameter yang digunakan:

Home directory
[tex]\texttt{--home /home/toor123}[/tex]Shell
[tex]\texttt{--shell \$(which bash)}[/tex]User ID
[tex]\texttt{--uid 110}[/tex]Group ID
[tex]\texttt{--gid 110}[/tex]Komentar
[tex]\texttt{--gecos "user ini baru lho"}[/tex]Username: [tex]\tt tool[/tex]

Asumsi: group dengan GID=110 belum ada.

Dengan adduser, kita harus membuat groupnya dengan [tex]\texttt{adduser --group --gid 110 tool}[/tex] atau [tex]\texttt{addgroup --gid 110 tool}[/tex]. Pakai saja nama group yang sama dengan nama user.

Kemudian, karena adduser akan menanyakan password untuk user yang dibuat, jika ingin user ini tanpa password, kita tambahkan parameter [tex]\texttt{--disabled-password}[/tex].

Jadi, perintah lengkapnya adalah:
addgroup --gid 110 tool && adduser --home /home/toor123 --shell $(which bash) --uid 110 --gid 110 --gecos "user ini baru lho" --disabled-password tool

Perintah tersebut harus dijalankan sebagai root. Jika bukan root, tambahkan sudo di depan perintah.

Outputnya kira-kira seperti ini:

[tex]\begin{aligned}&\texttt{Adding group 'tool' (GID 110) ...}\\&\tt Done.\\&\texttt{Adding user 'tool' ...}\\&\texttt{Adding new user 'tool' (110) with group 'tool' ...}\\&\texttt{Creating home directory '/home/toor123' ...}\\&\texttt{Copying files from '/etc/skel' ...}\end{aligned}[/tex]

______________
Cara 2: Dengan perintah “useradd”

Parameter yang digunakan:

Home directory
[tex]\texttt{--home-dir /home/toor123}[/tex] atau [tex]\texttt{-d /home/toor123}[/tex]
Perlu menambahkan parameter [tex]\texttt{--create-home}[/tex] atau [tex]\texttt{-m}[/tex] agar home directory dibuat jika belum ada.Shell
[tex]\texttt{--shell \$(which bash)}[/tex] atau [tex]\texttt{-s \$(which bash)}[/tex]User ID
[tex]\texttt{--uid 110}[/tex] atau [tex]\texttt{-u 110}[/tex]Group ID
Jika GID 110 sudah ada, kita gunakan parameter [tex]\texttt{--gid 110}[/tex] atau [tex]\texttt{-g 110}[/tex].
Namun, dengan asumsi GID 110 belum ada, kita gunakan parameter [tex]\texttt{--user-group}[/tex] atau [tex]\texttt{-U}[/tex]. Dengan parameter ini, useradd akan membuat group baru dengan GID=UID dan nama group sama dengan nama user.Komentar
[tex]\texttt{--comment "user ini baru lho"}[/tex] atau  [tex]\texttt{-c "user ini baru lho"}[/tex]Username: [tex]\tt tool[/tex]

Perintah lengkapnya adalah:
useradd --create-home --home-dir /home/toor123 --shell $(which bash) --uid 110 --user-group --comment "user ini baru lho" tool

Versi pendeknya:
useradd -m -d /home/toor123 -s $(which bash) -u 110 -U --comment "user ini baru lho" tool

Perintah tersebut harus dijalankan sebagai root. Jika bukan root, kita harus menambahkan sudo di depan perintah.

Jika sukses, tidak ada output yang ditampilkan.
