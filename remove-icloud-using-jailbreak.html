33. Buat Rumus Simple Present Tense- Using Verb- Using Be​


Jawaban:

Rumus:

Using Verb (Verbal)

(+) S + verb-1+O

(-) S + auxiliary (do/does) + not + bare infinitive

(?) do/does + S + bare infinitive

Using Be (Nominal)

(+) S+ be (am/is/are) +...

(-) S + be (am/is/are) + not+ ..

(?) be (am/is/are) + S+..

Penjelasan:

Simple present tense adalah suatu bentuk kata kerja untuk menyatakan fakta, kebiasaan, atau kejadian yang terjadi pada saat ini.
