17. 4.Apakah yang Anda ketahui dengan aplikasi berbasis cloud (cloud based application) ? 5. Apakah keuntungan dari penggunaan apkikasi berbasis cloud ?


Aplikasi berbasis cloud (cloud based application) adalah aplikasi yang disediakan oleh perusahaan penyedia layanan cloud. Maksud dari layanan cloud adalah aplikasi yang kita gunakan berada di internet dan tidak terinstall di komputer kita. Untuk mengakses aplikasi tersebut kita memerlukan akses internet.

Keuntungan dari penggunaan aplikasi berbasis cloud adalah keamanan data dan kita tidak dipusingkan untuk melakukan perawatan terhadap aplikasi tersebut karena sudah menjadi tanggung jawab penyedia layanan cloud tersebut.

Pembahasan

Infrastructure as a Service contohnya adala Amazon EC2, Telkom Cloud.Platform as a Service contoh aplikasinya adalah Windows AzureSoftware as a Service aplikasi Office 365 merupakan contoh aplikasi cloud jenis Software as a Service

Penggunaan aplikasi cloud tentu sangat membantu seorang pengusaha, karena bisa lebih fokus untuk pengembangan usahanya dan tidak dipusingkan dengan mengurus komputer dan aplikasi-aplikasi pendukungnya. Selain contoh-contoh diatas, masih ada contoh aplikasi cloud yang sering kita gunakan yaitu email dan media sosial.

18. 11. Komponen dari dua data atau lebih infrastruktur cloud disebut... a. Hybrid cloud b. Community cloud c. On-site-cloud d. Private cloud e. Public cloud​


Jawaban:

A. Hybrid cloud

Penjelasan:

Hybrid cloud adalah jenis infrastruktur cloud yang terdiri dari dua atau lebih komponen dari jenis infrastruktur cloud yang berbeda, seperti public cloud, private cloud, atau on-premises cloud. Dalam hybrid cloud, setiap jenis infrastruktur cloud tetap terpisah, tetapi dihubungkan melalui jaringan yang aman dan dapat berkomunikasi dengan lancar satu sama lain. Dengan menggunakan hybrid cloud, organisasi dapat memanfaatkan keuntungan dari setiap jenis infrastruktur cloud yang tersedia, misalnya keamanan dan kontrol yang lebih baik dari private cloud dan skalabilitas dan elastisitas dari public cloud. Hybrid cloud juga memberikan fleksibilitas bagi organisasi untuk memilih infrastruktur cloud yang paling cocok dengan kebutuhan bisnis mereka serta memungkinkan organisasi untuk memindahkan data dan aplikasi dari satu infrastruktur cloud ke infrastruktur cloud lainnya jika diperlukan.

29. Berapa Max Dpi dari Redmi Note 10S?Apakah 900/800 Dpi aman bagi Redmi Note 10S mohon penjelasan secara detail ​


Jawaban:

Redmi Note 10S memiliki resolusi layar Full HD+ 1080 x 2400 piksel dengan ukuran layar 6.43 inci. Namun, perlu diingat bahwa DPI (dots per inch) bukanlah sebuah spesifikasi yang terkait dengan perangkat itu sendiri, melainkan parameter yang berkaitan dengan output gambar yang dihasilkan oleh perangkat tersebut.

DPI adalah ukuran resolusi yang digunakan pada gambar cetak, bukan pada gambar digital yang ditampilkan pada layar perangkat seperti ponsel. DPI menunjukkan jumlah titik atau dot yang dapat ditempatkan dalam satu inci pada gambar cetak. Sedangkan untuk gambar digital, DPI biasanya digunakan untuk menghitung ukuran gambar dalam piksel per inci (PPI), yang menunjukkan jumlah piksel dalam satu inci pada layar.

Dalam konteks ini, DPI pada Redmi Note 10S tidak memiliki batasan tertentu. Namun, semakin tinggi DPI yang digunakan, semakin jelas dan tajam gambar yang dihasilkan, namun juga membutuhkan lebih banyak daya dan memperbesar ukuran file. Oleh karena itu, DPI yang digunakan sebaiknya disesuaikan dengan kebutuhan dan preferensi pengguna.

Untuk penggunaan normal, DPI sekitar 300-400 biasanya sudah cukup untuk memberikan gambar yang jelas pada layar Redmi Note 10S. DPI yang lebih tinggi seperti 900/800 dapat menghasilkan gambar yang lebih tajam, namun juga membutuhkan lebih banyak daya dan bisa membuat ukuran teks dan ikon menjadi lebih kecil. Pengguna dapat menyesuaikan DPI pada pengaturan tampilan perangkat Android, namun harus diingat bahwa semakin tinggi DPI yang digunakan, semakin besar kemungkinan penggunaan daya baterai yang lebih cepat dan bisa mempengaruhi kinerja perangkat.

38. private cloud public cloud hybrid cloud merupakan​


Jawaban:

Public cloud adalah cloud computing yang dikirimkan melalui internet dan dibagikan ke seluruh organisasi atau perusahaan. Private cloud adalah cloud computing yang didedikasikan hanya untuk organisasi atau perusahaan Anda. Hybrid cloud merupakan gabungan antara public cloud maupun private cloud.


39. sebutkan perbedaan penyimpanan cloud dengan non cloud​


Perbedaannya hanyalah pada bentuk fisik dan non fisik. Cloud merupakan bentuk off premise (non fisik) dari penyimpanan data melalui internet. Sedangkan data center adalah bentuk on premise (fisik) penyimpanan data pada perangkat keras.
