11. 1.Bagaimana cara menghapus latar belakang dari sebuah gambar pada Microsoft Word menggunakan format toool remove background .​


1.Pilih gambar yang ingin Anda hapus latar belakangnya.

2.Pada toolbar, pilih Format Gambar > Hapus Latar Belakang, atau Format > Hapus Latar Belakang.

Pada tab Format Gambar di pita toolbar, pilih opsi Hapus Latar Belakang.

Jika Anda tidak melihat Hapus Latar Belakang, pastikan Anda telah memilih gambar. Anda mungkin harus mengklik ganda gambar untuk memilihnya dan membuka tab Format Gambar .

3.Area latar belakang default akan berwarna magenta untuk menanda penghapusan, sementara latar depan akan mempertahankan warna alaminya.

4.Jika area default tidak benar, masuk ke Penghapusan Latar Belakang > Alat Gambar dan lakukan salah satu atau kedua hal berikut ini :

. jika bagian gambar yang ingin Anda simpan adalah magenta (ditandai untuk dihapus), pilih Tandai Area untuk Dipertahankan dan gunakan pensil gambar bentuk bebas Pensil yang digunakan untuk menggambar tabel untuk menandai area pada gambar yang ingin Anda simpan.

.untuk menghapus lebih banyak bagian gambar, pilih Tandai Area untuk Dihapus dan gunakan Pensil yang digunakan untuk menggambar tabel pensil gambar untuk menandai area tersebut.

5.Setelah selesai, pilih Simpan Perubahan atau Hapus Semua Perubahan.

Untuk menyimpan gambar dalam file terpisah untuk digunakan nanti, klik kanan gambar dan pilih Simpan sebagai Gambar

15. Tuliskan 4 macam page view dalam Microsoft word beserta fungsinya ​


Jawaban:

Read Mode.

Read Mode pada group view adalah tampilan membaca dokumen word dengan desain seperti membaca sebuah buku. Dengan mode ini, maka akan melihat dokumen dengan tampilan penuh, dengan setiap halaman yang digeser ke samping.

Print Layout.

Print Layout pada group view ber-fungsi untuk melihat dokumen dengan tampilan yang sama dengan hasil print. Mode ini sebenarnya kita jumpai saat kita membuka program word.

Dimana kita bisa melakukan editing secara langsung. Umumnya tampilan berbentuk vertikal dengan halaman digeser ke bawah.

Web Layout.

Web layout dalam group view digunakan untuk menampilkan dokumen dalam mode website. Dengan menggunakan mode ini, maka dokumen akan tampil sangat luas, tanpa batas. Tak memiliki halaman.

Draft.

Draft pada group view digunakan untuk menampilkan dokumen ke dalam mode Draft, membuat konsep. Hal ini biasnaya digunakan untuk penulisan buku. Dengan menuliskan list konsep maka akan mempermudah pembuatan buku.

Outline.

Outline dalam group view berguna untuk menuliskan Outline atau garis besar serta ringkasan dokumen yang dikerjakan.

26. Tab view Microsoft Word 2007 di kelompokan terdapat pada



TAB MENU HOME
Pada bagian ini anda dapat mengatur tampilan secara umum pada lembar kerja, seperti jenis huruf, clipboard, dan format sel lainnya.

    Terdapat 7 (tujuh) grup menu didalamnya, yaitu :
1.Clipboard didalamnya terdapat beberapa tombol yang bisa anda      gunakan untuk melakukan
2. copy, cut dan paste.
3. Font, bisa anda gunakan untuk mengatur format huruf seperti jenis, ukuran dan warna huruf.
4. Alignment, bisa anda gunakan untuk mengatur perataan teks yang ada pada sel.
5. Number, digunakan untuk mengatur data pada sel yang bertife angka.
6. Styles, digunakan untuk mengatur tampilan atau model pada tabel dan sel.
7. Cells, digunakan untuk menambah dan menghapus sel, serta untuk mengatur format sel dan sheet. Editing, bisa anda gunakan untuk menyeleksi data pada tabel.

 TAB MENU INSERT
Pada bagian ini anda dapat menyisipkan dan menambahkan obyek lain kedalam lembar kerja seperti gambar, shape, grafik dan wordart.
Shapes, gruip menu ini menampung berbagai macam bentuk shapes, maulai dari garis, kurva, hingga bentuk berbagai segi seperti bintang segi delapan, callout, dan yang lainnya.Tables, digunakan untuk mengelola data dengan model pivot tabel. Illustrations, dapat anda gunakan untuk mengubah gambar, grafik, dan smartart seperti struktur organisasi dan yang lainnya. Charts, dapat anda gunakan untuk mengubah grafik.Links, dapat anda gunakan untuk menghubungkan lembar kerja ketempat lain atau membuat link ketempat yang lain Text, dapat anda gunakan untuk membuat textbox, wordart, symbol, dan yang lainnya.

TAB MENU PAGE LAYOUT
Pada bagian ini anda dapat mengatur bentuk dan ukuran halaman pada workbook dan worksheet.
Terdapat 5 (lima) grup menu, yaitu :
Themes, digunakan untuk mengubah desain document secara keseluruhan, seperti warna, jenis huruf, dan lain-lain.Page setup, dugunakan untuk mengatur ukuran serta batas dari workbook dan worksheet.Scale to fit, digunakan untuk mengatur opsi pencetakan workbook. Sheet options, digunakan untuk mengatur properti yang ada pada sheet seperti grid dan headings (nama kolom).Arrange, digunakan untuk mengatur tampilan obyek yang ada pada sebuah worksheet.

TAB MENU FORMULAS
Didalamnya terdapat kumpulan grup menu yang digunakan melakukan pengolahan data mengunakan formula.
Didalam tab menu (formulas) terdapat 4 (empat) grup menu, yaitu :
Function Library, berisi berbagai kelompok fungsi yang ada pada excel. Setiap fungsi dikelompokkan kedalam satu kelompok dengan fungsi yang sama.Named cells, dapat anda gunakan untuk mengatur penamaan sel seperti range.Formula auditing, digunakan untuk melakukan pelacakan terhadap kesalahan fungsi dan rumus yang diguanakan.Calculation, digunakan untuk melakukan pengaturan perhitungan rumus atau fungsi yang anda gunakan pada sel.

TAB MENU DATA
Pada tab menu ini anda dapat mengatur penggunaan data pada worksheet. Termasuk mengelola data yang berasal dari sumber diluar excel.
Terdapat 5 (lima) grup menu pada tab menu (data), yaitu :
Get external data, dapat anda gunakan untuk mengambil data yang berasal dari luar excel.
